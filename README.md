# AspectYoungMel

Replaces Melinoe aspects with aspects of Young Melinoe in their stead.

## Features

Axe Aspect of Young Melinoe - Replaces the special with Block from the Early Access. 

## Known Issues

- This mod replaces Aspect of Melinoe instead of only adding a new aspect
- New Axe texture to match the icon
- Hammer 'Siege Shredder' creates 3 blocks with the AxeSpecial animation

## Installation

Use r2modman by ebkr from Thunderstore or GitHub.

While the mod has been tested decently well it is recommended to backup your save from %USERPROFILE%\Saved Games\Hades II\Profile*.sav in case there are issues.
